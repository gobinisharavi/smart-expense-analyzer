# Smart Expense Analyzer 

A Python-based data analysis tool that processes expense records and generates AI-style financial insights.

# Features
- 📊 Category-wise spending breakdown
- 📅 Monthly spending summary
- ⚠️ High expense detection (threshold-based flagging)
- 🤖 Automated AI insights (pattern recognition & recommendations)
- 📋 Clean formatted report output

# Tech Stack
- **Language:** Python 3
- **Concepts Used:** CSV parsing, data aggregation, algorithmic insight generation, OOP-style modular design
